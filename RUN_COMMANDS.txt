Windows Commands
================

Run these commands from the project root:

1. Create virtual environment
python -m venv venv

2. Activate virtual environment in PowerShell
.\venv\Scripts\Activate.ps1

If PowerShell blocks activation, run this once:
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

3. Install requirements
python -m pip install --upgrade pip
pip install -r requirements.txt

Shortcut setup
setup_windows.bat

4. Download and prepare real public dataset
python src\prepare_dataset.py --download --reset

Shortcut dataset preparation
prepare_dataset.bat

5. Manual dataset option
Place images in:
dataset\train\mask
dataset\train\no_mask
dataset\train\incorrect_mask
dataset\val\mask
dataset\val\no_mask
dataset\val\incorrect_mask
dataset\test\mask
dataset\test\no_mask
dataset\test\incorrect_mask

For a 2-class dataset, use only mask and no_mask folders.

6. Train model
python src\train_cached.py --epochs 60 --class_weight_mode sqrt --reuse_features --occlusion_negatives_per_image 2

Full transfer-learning alternative
python src\train.py

Shortcut training
train_model.bat

7. Evaluate model
python src\evaluate.py --split test

8. Predict one image
python src\predict_image.py sample_images\test.jpg

9. Webcam detection
python src\webcam_detect.py

Shortcut webcam
run_webcam.bat

10. Streamlit web app
streamlit run src\streamlit_app.py

Shortcut Streamlit
run_streamlit.bat
